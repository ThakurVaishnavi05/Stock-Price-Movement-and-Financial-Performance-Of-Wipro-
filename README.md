# Wipro Stock Price Movement and Financial Performance Analysis Using Time Series Techniques

## Project Overview

This project examines the historical stock price movement and financial performance of Wipro Ltd. through the application of time series analysis and financial ratio analysis. The study evaluates stock price trends, volatility patterns, forecasting outcomes, and key financial indicators to understand the relationship between market performance and business growth.

## Objectives

- Analyze the historical stock price movement of Wipro Ltd.
- Identify trends, fluctuations, and volatility in stock prices.
- Forecast short-term stock price movements using time series techniques.
- Evaluate the company's financial performance through profitability, liquidity, and solvency ratios.
- Examine the relationship between stock price movement and financial performance indicators.

## Data Sources

The study is based on secondary data collected from:

- Historical stock price data of Wipro Ltd.
- Annual reports and financial statements.
- Publicly available financial and stock market databases.

## Methodology

### Stock Price Analysis
- Descriptive Statistical Analysis
- Trend Analysis
- Volatility Analysis
- Monthly Return Analysis
- Moving Average Forecasting

### Financial Performance Analysis
- Profitability Ratio Analysis
- Liquidity Ratio Analysis
- Solvency Ratio Analysis
- Revenue and Net Profit Trend Analysis

### Relationship Analysis
- Correlation Analysis between stock prices and financial performance indicators.

## Tools Used

- Microsoft Excel
- Time Series Analysis Techniques
- Statistical Analysis
- Financial Ratio Analysis
- Data Visualization

## Key Findings

- Wipro demonstrated a positive long-term stock price trend during the study period.
- Financial performance indicators reflected operational stability and consistent growth.
- Forecasting techniques provided insights into short-term stock price behavior.
- Financial performance and stock price movement showed a measurable relationship.
- The combined use of time series analysis and financial analysis supports informed investment evaluation.
